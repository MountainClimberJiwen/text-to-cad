TEMPLATE: "full_station"
Description: Complete pick-and-place station with vibration feeder, gantry, and fixture.
Parameters:
- station_width (mm): Overall base plate width. Default: 800
- station_depth (mm): Overall base plate depth. Default: 500
- bowl_diameter (mm): Vibration bowl outer diameter. Default: 200
- gantry_span (mm): Distance between gantry columns. Default: 300
- gantry_height (mm): Column height from base to beam bottom. Default: 250
- vertical_stroke (mm): Vertical cylinder travel. Default: 120
- horizontal_stroke (mm): Horizontal cylinder travel. Default: 250
- gripper_span (mm): Gap between gripper jaws when open. Default: 20
- vib_x, vib_y (mm): Approximate position of vibration feeder relative to station origin.
                     These are HINTS, not exact coordinates.
- fix_has_push (bool): Whether fixture has a push cylinder. Default: true

Engineering guarantees (template enforces these automatically):
- Gantry beam is ALWAYS supported by TWO columns (portal frame, never cantilever).
- Columns ALWAYS sit on the base plate.
- Beam width ALWAYS covers column span + 20mm margin.
- Vertical cylinder stroke > 50mm ALWAYS gets guide rails (auto-added).
- Gripper jaw volume is ALWAYS >= 0.1% of base plate volume (auto-scaled if needed).
- Hopper is ALWAYS independently supported, never on vibrating bowl.
