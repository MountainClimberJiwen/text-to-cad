# 05 详细机构设计模块（核心）

## 模块目标

按工序完成执行机构、检测机构与定位机构的细化设计，保障装配精度与良率。

## 当前设计思路

- 逐工序设计执行 + 检测 + 定位三类机构
- 以“动作可执行、状态可检测、异常可隔离”为原则

### 子模块摘要

- PFS 上料：人工放蜂巢盒 -> 机械手搬运 -> 双相机检测方向/垂直度 -> 空盒自动排出
- 激发/收针组件上料：人工放 Tray -> 载座精准定位 -> 弹性夹紧
- 激发组件装入 PFS：传感定位 -> 气缸 + 夹爪压装
- 固定插片上料与检测：振动盘上料 -> 导向定位 -> 气缸压装 -> 位移检测
- PFS 插入收针组件：气缸移载压装 -> 棱镜 + 相机双卡扣检测
- 激发套筒装配：二次定位 -> 夹爪撑开 -> 收针组件装入 -> 卡扣检测
- 套筒压入外壳：预装配 -> 伺服电缸压装 -> 压力位移曲线 -> 双产品兼容
- 顶盖装配：振动盘上料 -> 移载压装 -> 卡扣检测
- 贴标：贴标机出标 -> 伺服旋转滚贴 -> 相机检测位置
- OK/NG 下料：机构分流 -> OK 输送线，NG 废料口

## 可扩展操作清单

1. 为每个子模块补齐动作时序图（含互锁条件）
2. 定义传感器 I/O 点位与信号命名
3. 建立压装力/位移判定阈值与报警等级
4. 输出关键治具公差链分析

## 输入/输出

- 输入：流程定义、产品尺寸链、节拍目标
- 输出：机构方案包、时序图、I/O 清单、检测判定规则

## 关键约束

- 关键压装动作必须可追溯（力-位移曲线）
- 检测策略必须可覆盖错装/漏装/未到位

## 风险与检查点

- 风险：动作节拍冲突导致卡站
- 检查点：是否完成单工位仿真与节拍平衡
