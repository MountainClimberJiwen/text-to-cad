# 02 产品工艺流程模块

## 模块目标

定义完整工序链路、工位动作顺序与人工/自动职责边界。

## 当前设计思路

- 总流程：人工上料 -> 自动装配 -> 自动检测 -> 压合 -> 顶盖装配 -> 贴标 -> OK/NG 下料
- 动作拆分：PFS 上料、激发组件装配、固定插片装配、收针组件装配、套筒压入、顶盖装配、贴标、检测分流
- 原则：人工辅助上料 + 自动执行装配检测，减少关键质量环节的人为波动

## 可扩展操作清单

1. 输出工艺流程图（BPMN/泳道图）
2. 定义每工序 CT（Cycle Time）与瓶颈工位
3. 增加异常分支（缺件、错装、检测 NG）
4. 建立工序-设备动作映射表

## 输入/输出

- 输入：需求基线、产品 BOM、质量要求
- 输出：流程图、工位动作表、异常处理流程

## 关键约束

- 工艺顺序不得破坏装配定位基准
- 检测节点必须覆盖关键质量特性

## 风险与检查点

- 风险：流程与机构动作脱节导致节拍失真
- 检查点：是否完成工序级 CT 和异常闭环定义
