# 03 物料模块

## 模块目标

明确物料清单、上料方式与工序匹配关系，避免错装漏装。

## 当前设计思路

- 关键物料：PFS、激发组件（2 款）、固定插片、收针组件、激发套筒、外壳组件（2 款）、顶盖
- 工序匹配：每道装配工位物料唯一，确保可追溯
- 上料策略：区分人工上料与振动盘自动上料

## 可扩展操作清单

1. 补全物料编码与版本规则
2. 建立物料-工位-治具三向映射
3. 定义补料节拍与最小库存预警
4. 增加混料防错机制（颜色、传感、视觉）

## 输入/输出

- 输入：BOM、供应商数据、工位设计
- 输出：物料清单、上料方式表、防错策略表

## 关键约束

- 双型号共线时必须支持快速切换与防混
- 关键件需满足批次追溯要求

## 风险与检查点

- 风险：同类件外观接近造成误投料
- 检查点：是否建立混料预防 + 检测双保险
