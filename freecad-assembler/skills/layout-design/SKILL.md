# Layout Design Skill

用于布局和上料结构设计，输出建议包含：
1. 工序拆解（上料 -> 定位 -> 装配/检测 -> 流转）
2. 机构选型（供料/输送/定位）
3. 动作逻辑与节拍校核
4. 布局出图与安全边界
