# 自动上料装配单元

**功能用途**：通过伺服电机驱动滚筒输送工件，气动推料机构将工件推送至下一工位，实现自动化上料与定位。

========== 一、全局基准定义 ==========
- 基准安装面：机架顶面（Z=800mm 安装板顶面）
- 产线中心轴线：机架几何中心 X=0, Y=0
- 坐标原点(0,0,0)：机架底面中心
- Z轴方向：垂直向上

========== 二、第三方设备清单 ==========
1. PLC 控制器
   品牌型号：西门子 S7-1200
   外形尺寸(L×W×H)：140×100×60mm
   安装孔参数：4×φ5mm，孔距 120×80mm
   3D模型引用编号：plc_module.step

2. 伺服电机
   品牌型号：松下 MHMF042
   外形尺寸：法兰φ80mm，总长~120mm
   安装孔参数：4×φ6mm，圆周中心距φ60mm
   3D模型引用编号：servo_motor.step

3. 气泵
   品牌型号：贝克 VT4.4
   外形尺寸(L×W×H)：300×200×180mm
   安装孔参数：4×φ8mm，孔距 280×180mm
   3D模型引用编号：air_pump.step

========== 三、机械装配定位要求 ==========
1. PLC 安装
   贴合基准面：机架顶面安装板顶面（Z=800mm）
   中心坐标：X=500mm, Y=0mm, Z=830mm
   形位约束：底面贴合，水平无扭曲
   固定螺栓规格：4×M5，锁紧力矩 8 N·m

2. 伺服电机安装
   贴合基准面：滚筒左端支架外侧面
   中心坐标：X=-320mm, Y=0mm, Z=600mm
   形位约束：法兰端面贴合支架，电机输出轴与滚筒同轴度≤0.05mm
   固定螺栓规格：4×M6，锁紧力矩 12 N·m

3. 气泵安装
   贴合基准面：机架左侧辅助安装平台顶面（Z=750mm）
   中心坐标：X=-400mm, Y=0mm, Z=840mm
   形位约束：底座全贴合，整机水平放置
   固定螺栓规格：4×M8，锁紧力矩 20 N·m

4. 气缸安装（推料执行机构）
   贴合基准面：机架前端安装板
   中心坐标：X=300mm, Y=-350mm, Z=810mm
   形位约束：活塞杆运动方向与产线 Y 轴平行
   固定螺栓规格：2×M6

5. 电磁阀安装
   贴合基准面：机架顶面
   中心坐标：X=-300mm, Y=200mm, Z=815mm
   形位约束：底面贴合，气口朝向便于管路连接

========== 四、电气系统接驳 ==========
- PLC Q0.0 → 伺服驱动器脉冲信号
- PLC Q0.1 → 电磁阀控制信号
- 线槽安装高度：Z=780mm，沿机架后沿布置
- 线缆规格：动力线 1.5mm²，信号线 0.5mm²屏蔽线

========== 五、气动系统接驳 ==========
- 气泵出口 φ20mm → φ16 气管 → 电磁阀进气口 P
- 电磁阀出气口 A → φ8 气管 → 气缸前腔
- 电磁阀出气口 B → φ8 气管 → 气缸后腔
- 工作气压范围：0.4~0.6 MPa

========== 六、工艺&验收要求 ==========
- 锁紧力矩标准：PLC M5=8N·m / 电机 M6=12N·m / 气泵 M8=20N·m
- 关键配合公差：电机轴与滚筒同轴度≤0.05mm；安装孔 H7
- 整体干涉检查：所有设备间隙≥5mm（本设计最小间隙约 50mm）
- 现场装配验收指标：设备水平度≤0.1mm/m；紧固件无松动；气管无泄漏
