# Industrial CAD Skill

工业自动化产线 CAD 装配设计 Skill，为 coding agents 提供标准化的自动化设备描述、产线装配模板和工程约束规范。

## What It Can Do

- 定义工业自动化产线的**全局基准坐标系**和**安装基准**。
- 引用**标准第三方设备台账**：PLC、伺服电机、气泵、气缸、传感器、直线滑台等。
- 应用**机械装配约束**：贴合、同轴、平行、垂直、中心对齐、面距离。
- 规范**电气/气动接口**位置、规格和管路走向。
- 提供**产线装配描述模板**，确保输出结构统一、可直接用于 CAD 建模落地。
- 定义**紧固件力矩、加工公差、间隙规范、验收标准**。
